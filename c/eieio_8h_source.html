56 // Bitfields go from LSB to MSB; SpiNNaker is little-endian, and making them all
57 // use uint16_t as a base type means that we know they'll all be in the same
58 // memory slot (which they will fit into).
59 //
60 // http://infocenter.arm.com/help/index.jsp?topic=/com.arm.doc.faqs/ka10202.html
61 // http://infocenter.arm.com/help/index.jsp?topic=/com.arm.doc.dui0491e/Babjddhe.html
62 //
63 // THIS IS NOT PORTABLE TO NON-SPINNAKER!
